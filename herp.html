Herp

For my internship, I worked for Joutsan Hautauspalvelu OY making a web based enterprise resource planning software to help them keep track of their tasks and assignments.

The backend was made with Django framework and deployed in AWS using Docker containers.

As a thesis, I focused on developing easy-to-use user interface for the software using the React library. A lot of emphasis was put on making the interface accessible by following modern accessibility standards.
